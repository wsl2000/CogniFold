Design philosophy

Memory consolidates — it doesn't search.

Human memory doesn't look things up — it settles. Borrowing the brain's complementary learning systems, CogniFold writes events down fast, replays what recurs into durable concepts, and turns that understanding into goals. Nothing is fetched on demand: structure emerges under the stream.

But consolidation is only the starting mechanism. Our larger goal is to bring the full landscape of human memory-systems research into working code — modeling every documented system and seeing how far a single substrate can go. Nineteen are on the map today; ~53% covered.
